## Работа с API системы защиты информации (VirusTotal)

### Описание

В рамках данной лабораторной работы реализован Python-скрипт для взаимодействия с API сервиса VirusTotal.

Скрипт:
- выполняет авторизацию через API-ключ;
- отправляет GET-запрос к API v3;
- получает JSON-ответ по SHA-256 хэшу файла;
- выводит полный JSON в консоль;
- выводит краткую статистику анализа (last_analysis_stats);
- сохраняет ответ в файл `response.json`.

Используется бесплатный API-ключ VirusTotal.

- json (встроенная библиотека Python)
